# Watchfinder

AI-powered luxury watch identification and analysis using computer vision.

## What it does

Watchfinder uses advanced computer vision and AI to identify luxury watches from photos, providing detailed analysis of make, model, and specifications. Upload an image of any luxury timepiece and get instant recognition with comprehensive information about the watch, its market value, and historical context.

## Tech Stack

- **Next.js** - React framework for production
- **TypeScript** - Type-safe development
- **Supabase** - Backend database and authentication
- **Tailwind CSS** - Utility-first styling
- **shadcn/ui** - Accessible component library

## Getting Started

```bash
git clone https://github.com/1aday/Watchfinder.git
cd Watchfinder
npm install
npm run dev
```

## Environment Variables

Create a `.env.local` file with the following:

```
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_supabase_anon_key
OPENAI_API_KEY=your_openai_api_key
```
